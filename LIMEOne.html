🔔 外部連携 (Google カレンダー / LINE 通知)
📡 新規案件登録時・回収予定日登録時 に Google Apps Script (GAS) 経由で:
📅 ライム共有カレンダーへ予定追加 / 📨 スタッフグループ LINE へ通知
初回のみ GAS スクリプトの作成と URL 登録が必要です。詳細は下の「📖 セットアップ手順を見る」をクリック。
空欄なら外部連携 OFF。URL を設定するだけで通知が動きます。
GAS 側に Calendar ID / LINE Token / LINE Group ID を設定する必要があります。
テスト押下でカレンダー+LINEの両方が動くか確認できます。
📖 セットアップ手順を見る (初回のみ)
【1】LINE 公式アカウント作成
① LINE Developers でログイン → Provider 作成 (ライム等) → Messaging API channel 作成
② Channel access token (long-lived) を発行・コピー (後でGASに貼る)
③ 作成された公式アカウントを 店舗スタッフグループ LINE に招待
④ グループ ID を取得 (手順: LINE Official Account Manager の「グループ管理」または webhook で確認)

【2】Google Apps Script 作成
① script.new で新規プロジェクト作成 (名前: ライム通知 等)
② 下記の 「📋 GASコードをコピー」 ボタンでコードをコピーして貼り付け
③ メニュー: プロジェクト設定 (歯車) → スクリプトプロパティに以下を追加:
　　・CALENDAR_ID = ライム共有カレンダーの ID (カレンダー設定→カレンダーの統合)
　　・LINE_TOKEN = ①で取得した Channel access token
　　・LINE_GROUP_ID = ④で取得したグループ ID
　　・EFAX_DOMAIN (任意 / モバイル版でFAX送信する場合) = eFax受信用ドメイン (既定: efaxsend.com)
④ デプロイ → 新しいデプロイ → 種類: ウェブアプリ → 実行ユーザー: 自分 / アクセス: 全員 → デプロイ
⑤ 表示された ウェブアプリ URL をコピー

【3】このシステムに URL を貼る
① 上の 「Google Apps Script Web App URL」 欄に URL を貼り付け
② 通知トリガをチェック

🔢 受付番号 採番ルール
📋 運用方針
・Excelから取り込んだ既存番号は変更しません（手作業で付けた番号も保持）
・システムから新規受付した時のみ自動採番します

自動採番フォーマット: YYYY-MM-NNNN
・YYYY = 年度 (4月始まり。1〜3月受付は前年度扱い)
・MM = 受付月 (01〜12)
・NNNN = 年度ごとの連番 (新年度の最初の受付は 0001 から)

採番例
・2026/4/15 受付 → 2026-04-0001 (2026年度スタート)
・2026/12/10 受付 → 2026-12-0050 (年度内通算50件目)
・2027/1/5 受付 → 2026-01-0080 (年度2026、受付月01)
・2027/3/31 受付 → 2026-03-0120 (年度2026最終日)
・2027/4/1 受付 → 2027-04-0001 (新年度リセット)
⚠️ 既存データを一括振り直し（通常は使用しません）
⚠️ 注意: このボタンを押すと Excelから取り込んだ既存番号も含めて全件 が新形式に振り直されます。通常運用では押さないでください。
過去番号を会計や保険会社へ提出済みの場合、整合性が崩れます。
※ 受付日順でソート → -0001から振り直し
